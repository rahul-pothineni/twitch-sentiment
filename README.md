# TwitchTell

An application that helps Twitch streamers and viewers analyze live stream chat data.  Helping users detect jumps/dips in chat sentiment in real time!

## 📦 Technologies 

- `PostgreSQL (Django ORM)`
- `Python (Django)`
- `Confluent Kafka (quixstreams)`
- `PostgreSQL`
- `React`
- `TypeScript`
